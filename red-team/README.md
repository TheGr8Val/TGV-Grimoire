# red-team/

> Adversary simulation, tabletop exercises, detection gap analysis, and hypothesis generation for offensive security practitioners

This module contains prompts for authorized red team and purple team operations. All prompts assume a legitimate engagement context: authorized pentest, red team exercise, CTF, or internal security research.

**These prompts are not for unauthorized access or malicious use.**

---

## Prompts

| File | Use case |
|------|----------|
| [ttx-scenario-builder.md](./ttx-scenario-builder.md) | Build a tabletop exercise scenario for a given threat actor and target org |
| [detection-gap-analysis.md](./detection-gap-analysis.md) | Identify TTPs with no current detection coverage |
| [custom-payload-obfuscation-brainstorm.md](./custom-payload-obfuscation-brainstorm.md) | Brainstorm obfuscation approaches for authorized engagements |

---

## Ethics and scope

Every prompt in this module includes a scope field. Fill it in accurately — it keeps the LLM output relevant to your actual engagement and serves as documentation that you operated within authorized boundaries.

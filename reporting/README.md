# reporting/

> Incident reporting, executive summaries, findings translation, and postmortem documentation

This module covers the communication layer of security work — turning technical findings, incident timelines, and raw notes into documents that stakeholders can act on.

---

## Prompts

| File | Use case |
|------|----------|
| [executive-summary.md](./executive-summary.md) | Translate a technical incident timeline into a non-technical executive brief |
| [findings-to-remediation.md](./findings-to-remediation.md) | Convert pentest findings into prioritized remediation tickets |
| [incident-postmortem.md](./incident-postmortem.md) | Generate a structured 5-whys postmortem from raw incident notes |

---

## [ES] Descripcion del modulo

Este modulo cubre la capa de comunicacion del trabajo de seguridad — convertir hallazgos tecnicos, cronologias de incidentes y notas brutas en documentos accionables para los stakeholders.

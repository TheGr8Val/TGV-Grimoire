# blue-team/

> Detection engineering, analyst playbooks, and defensive posture analysis

This module covers the operational and engineering side of defense: reviewing detection rules, building response playbooks, and mapping an organization's most critical assets to monitoring controls.

---

## Prompts

| File | Use case |
|------|----------|
| [detection-rule-review.md](./detection-rule-review.md) | Review an existing Sigma/KQL/SPL rule for logic gaps and false positive risk |
| [playbook-draft.md](./playbook-draft.md) | Generate a structured analyst response playbook for a given alert type |
| [crown-jewels-mapping.md](./crown-jewels-mapping.md) | Identify high-value targets and recommended monitoring controls for an org |

## [ES] Descripcion del modulo

Este modulo cubre el lado operacional e ingenieril de la defensa: revision de reglas de deteccion, construccion de playbooks de respuesta, y mapeo de activos criticos a controles de monitoreo.

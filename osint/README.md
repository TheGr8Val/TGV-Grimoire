# osint/

> Open-source intelligence prompts — passive recon, persona analysis, lookalike domain generation for monitoring

This module covers OSINT workflows: reconnaissance, persona investigation, and infrastructure monitoring. All prompts are designed for defensive use cases (monitoring, threat intel, fraud investigation) or authorized red team recon.

---

## Prompts

| File | Use case |
|------|----------|
| [passive-recon-checklist.md](./passive-recon-checklist.md) | Generate a structured OSINT recon checklist for a target domain or org |
| [persona-analysis.md](./persona-analysis.md) | Analyze a social media persona for indicators of inauthenticity |
| [domain-lookalike-gen.md](./domain-lookalike-gen.md) | Generate lookalike/typosquat domain variants for monitoring or brand protection |
